# Code of Conduct

This project adopts the Contributor Covenant Code of Conduct version 2.1.

- Be respectful and direct.
- Assume good intent and focus on technical facts.
- Report unacceptable behavior to the maintainers listed in [SECURITY.md](./SECURITY.md).

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
