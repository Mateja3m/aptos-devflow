# Quickstart

## Prerequisites

- Node.js 18.18 or newer
- npm 10 or newer
- Aptos CLI for Move compile checks

## Install

```bash
npm install
```

## Primary repository proof

```bash
npm run doctor
```

## Full repository verification

```bash
npm run verify
```

## Package proof flow

```bash
npm install @idoa/aptos-devflow-cli
npx devflow doctor
```

Fallback package execution:

```bash
npx @idoa/aptos-devflow-cli doctor
```
